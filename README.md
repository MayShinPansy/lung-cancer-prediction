# Lung Cancer Prediction

Machine learning project for lung cancer prediction using classification models.

---

## 🎯 Project Overview

This project focuses on predicting lung cancer outcomes using multiple machine learning models. The workflow includes data preprocessing, handling imbalanced data, and evaluating model performance using cross-validation techniques.

---

## 🤖 Models Used

* K-Nearest Neighbors (KNN)
* Logistic Regression
* Multi-Layer Perceptron (MLP)

---

## ⚙️ Techniques & Methods

* Data preprocessing and cleaning
* Feature scaling
* ADASYN for class imbalance handling
* K-Fold and Stratified K-Fold cross-validation
* Model performance evaluation

---

## 🛠 Tools & Technologies

* Python
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
